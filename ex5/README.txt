5 - Qual(is) a(s) vantagem(ns) de usar typescript ao invés de javascript puro?

Como já demonstrado no exercício 4, por typescript possuir tipagem estática opicional, a vantagem de ser utilizado ao invés de javascript é justamente essa. Como javascript é com tipagem dinâmica, podem ocorrer erros durante a produção que podem ser muito custosos durante a produção, gerando custos extras e retrabalhos. 
Além disso, typescript suporta mais conceitos de POO, como classes,
interfaces, herança, etc. POO é um paradigma que é muito mais organizado e quando o projeto cresce em tamanho e complexidade, essa organização se torna uma vantagem enorme.
